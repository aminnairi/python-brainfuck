# Contributing

## Requirements

- UNIX-like operating system
- Git
- Docker
- Docker Compose
- GNU/Make

## Installation

### HTTPS

```console
$ git clone https://github.com/aminnairi/python-brainfuck
$ cd python-brainfuck
```

### SSH

```console
$ git clone git@github.com:aminnairi/python-brainfuck
$ cd python-brainfuck
```

## Unit tests

```console
$ make unit_tests
```

## Integration tests

```console
$ make integration_tests
```
